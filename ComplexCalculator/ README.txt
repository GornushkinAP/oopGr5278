Калькулятор комплексных чисел
Этот проект представляет собой простой калькулятор для работы с комплексными числами. Проект состоит из трех основных частей: модели, представления и презентера.

Структура проекта
Model: Этот пакет содержит классы, отвечающие за выполнение операций над комплексными числами. 
Включает в себя класс ComplexCalcModel для выполнения операций сложения, умножения и деления.

View: Пакет содержит классы, отвечающие за взаимодействие с пользователем.
Включает в себя класс ComplexCalcView для отображения пользовательского интерфейса и взаимодействия с пользователем.

Presenter: Этот пакет содержит классы, которые связывают модель и представление.
Включает в себя класс ComplexCalcPresenter для обработки действий пользователя и передачи данных между моделью и представлением.

Запуск приложения
Для запуска приложения необходимо создать экземпляр модели ComplexCalcModel, представления ComplexCalcView и презентера ComplexCalcPresenter. 
Затем можно запустить пользовательский интерфейс для работы с комплексными числами.

Использование
При запуске приложения пользователю будет предложено выбрать операцию (сложение, умножение, деление) и ввести два комплексных числа через пробел. 
Результат операции будет отображен на экране. 

Так же производится логирование данных в текстовый файл calcLog.txt. После каждого нового вычисления происходит сохранение в файл.
